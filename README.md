# OfficeOneClick

🎯 一键下载、安装并激活 Microsoft Office！支持自定义安装、KMS 激活，适用于 Windows 用户的高效办公利器。

---

## 📦 功能简介

- ✅ 一键全自动安装 Office（含工具下载、配置、激活）
- 🔧 手动分步操作：  
  1. 下载 Office 安装工具  
  2. 下载 Office 安装包  
  3. 执行 Office 安装  
  4. 激活 Office  
  5. 卸载脚本与缓存文件（不影响 Office 使用）
